## Anchor

- Primary: Trigger.dev (https://trigger.dev) — run/activity dashboards, status timelines, operational proof surfaces.
- Secondary: BetterStack (https://betterstack.com) — monitoring-style incident/status clarity.

## Palette

- Background: #f7f8fb
- Surface: #ffffff
- Surface muted: #f1f4f8
- Border: #d9e0ea
- Text primary: #172033
- Text secondary: #637083
- Accent: #1463ff — used only for primary actions, active nav, and selected controls
- Success: #16845b
- Warning: #a15c00
- Destructive: #c9342f
- Code background: #101827

## Typography

- Display: system sans-serif 700 — used for page title and section headings only
- Body: system sans-serif 400/500/600 — used for app copy
- Mono: ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas — used for ids, hashes, and reason codes
- Scale: 12/13/14/16/20/24/32

## Spacing

- Base unit: 4px
- Scale in use: 4, 8, 12, 16, 20, 24, 32, 40

## Motion

- Hover: border-color and background-color shift, 140ms ease
- Page transition: none
- Loading: inline status text and stable skeleton rows

## Banned

- Purple/pink/blue gradient hero patterns
- Rounded-xl shadow-md generic cards
- Centered marketing hero as first screen
- Lorem ipsum, John Doe, Jane Smith, random fake users
- ui-avatars.com, randomuser.me, picsum.photos
- Text that explains how to use the UI instead of letting controls speak for themselves
