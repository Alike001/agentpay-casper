# CSPR.fans Community Voting Strategy

Status: Draft  
Goal: maximize qualification odds through the community voting path while still satisfying technical eligibility.

## Public Positioning

Short tagline:

> Let AI agents spend safely on Casper.

One-sentence pitch:

> Casper Agent Commerce Firewall lets autonomous AI agents pay for tools and services only inside user-defined rules, with every approved action recorded as a Casper Testnet receipt.

Plain-language explainer:

> AI agents are starting to spend money. We built the safety layer: set a budget, approve services, let the agent act, and block it when it tries to overspend.

## Why People Should Vote

Use simple claims:

- It is useful: agents need spending limits.
- It is real: it produces Casper Testnet transactions.
- It is safe: the AI cannot bypass policy.
- It is visual: you can see allowed and blocked actions.
- It helps Casper: it creates reusable infrastructure for agent commerce.

Avoid overly technical framing:

- Do not lead with "MCP gateway policy registry."
- Do not over-explain x402.
- Do not make it sound like a compliance spreadsheet.

## Vote Page Copy

> AgentSafe Casper

> AgentSafe Casper lets users create AI agents that can call paid services and write on-chain receipts, but only within explicit spending rules. The demo shows an agent buying an RWA risk report, recording the approved action on Casper Testnet, then getting blocked when it tries to overspend.

## Community Demo Flow

Keep it to three screens:

1. "Here is the agent budget."
2. "Here is the approved purchase and Casper receipt."
3. "Here is the blocked overspend."

Community voters do not need to understand every contract. They need to understand the before/after:

- Before: agent spending is risky.
- After: Casper gives rules and receipts.

## Final-Round Narrative

If we qualify, shift from catchy public messaging to judge-grade depth:

- how smart contracts enforce/commit state
- how MCP/x402/CSPR.cloud/Odra fit together
- why this can become Casper's agent-commerce safety layer
